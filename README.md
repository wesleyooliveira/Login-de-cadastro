### 🛠 Tecnologias
- **HTML5** - Estrutura semântica com `header`, `main`, `footer`
- **CSS3** - Flexbox, Media Queries, Mobile-First
- **Github Pages** - Deploy e hospedagem

### 📱 Features
- ✅ Layout Mobile-First responsivo
- ✅ Header profissional com Flexbox `space-between`
- ✅ Formulário centralizado com `align-items` e `justify-content`
- ✅ Design adaptável para 320px até 1920px
- ✅ Código semântico e acessível

### 🧠 O que aprendi
1. Estruturação HTML semântica
2. Domínio de Flexbox: `justify-content`, `align-items`, `flex-direction`
3. Responsividade com `@media` e breakpoints
4. Deploy estático com Github Pages
